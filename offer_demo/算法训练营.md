## 1.五步刷题法
```markdown
1.看题目讨论题目细节和解法。
2.能够有解法思路开始解题。
3.无解题思路，直接看题解，比较各个题解之间的优劣，寻找解题思路和优秀题解。
4.一天之后再一次解题。
5.一周之后再一次解题。
6.一月之后最后一次解题。
```
## 2.算法时间复杂度和空间复杂度
```markdown
递归的时间复杂度：
    int fib(int n){
        if(n<=2) return n;
        return fib(n-1)+fib(n-2);
    }
    //O(n^2)
```
